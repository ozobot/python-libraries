Welcome to Ozobot ORA API's documentation!
==========================================

Currently, the library provides a single module, :doc:`ozobot.ora.simple </simple>`, which
provides a simple interface to Ozobot ORA. The primary focus is on beginners and
robotics students whose primary goal is learning robotics.
